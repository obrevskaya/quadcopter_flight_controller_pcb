# Разработка платы контроллера полёта квадрокоптера

## Описание проекта

Проект посвящён разработке печатной платы контроллера полёта квадрокоптера в системе автоматизированного проектирования KiCad.

В рамках работы были выполнены:

* разработка структурной схемы устройства;
* выбор и обоснование элементной базы;
* разработка принципиальной электрической схемы;
* трассировка двухслойной печатной платы;
* создание трёхмерной модели печатной платы;
* подготовка производственной документации для изготовления платы.

---

## Используемая элементная база

### Микроконтроллер

**STM32F405VGTx**

* ARM Cortex-M4;
* 32-разрядная архитектура;
* обработка данных датчиков;
* формирование управляющих сигналов.

### Датчики

**ICM-20602**

* трёхосевой гироскоп;
* трёхосевой акселерометр.

**LIS3MDL**

* трёхосевой магнитометр;
* определение ориентации относительно магнитного поля Земли.

**BMP280**

* барометрический датчик давления;
* оценка высоты полёта.

### Навигация

**NEO-M8N**

* GPS/GLONASS модуль;
* определение координат и скорости движения.

### Подсистема питания

**LM7805**

* стабилизатор питания 5 В.

**AMS1117-3.3**

* стабилизатор питания 3.3 В.

## Разработка печатной платы

Печатная плата выполнена в системе автоматизированного проектирования **KiCad**.

Особенности реализации:

* двухслойная конструкция;
* заливка полигона земли на обоих слоях;
* разделение сигнальных и силовых цепей;
* минимизация длины соединений;
* размещение монтажных отверстий;
* подготовка Gerber-файлов для производства.

Размер платы:

```text
70 × 70 мм
```

В результате работы была разработана печатная плата контроллера полёта квадрокоптера, включающая:
